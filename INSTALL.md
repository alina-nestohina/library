# 🛠 Інструкція з розгортання та встановлення програми

### 1. Попередні вимоги до програмного забезпечення
Перед запуском програмної системи «Особиста бібліотека» на комп'ютері користувача має бути встановлено:
* Операційна система: Windows 10 / Windows 11
* Середовище виконання: .NET Desktop Runtime 6.0 (або новіша версія)

### 2. Покроковий опис дій зі встановлення програми
Продукт є портативним (Portable) і не потребує класичного інсталятора:
1. Завантажте архів із бінарними файлами програми `library.zip`.
2. Розпакуйте вміст архіву в будь-яку зручну локальну папку (наприклад, `C:\Programs\LibraryApp\`).
3. Переконайтеся, що в папці присутні файли `library.exe`, `library.dll` та конфігураційні залежності.
4. Запустіть програму, виконавши подвійний клік на файл `library.exe`.

### 3. Покроковий опис дій з видалення програми
Програма не вносить змін до системного реєстру ОС:
1. Закрийте графічне вікно програми.
2. Повністю видаліть папку з файлами програми (перемістіть її в Кошик).
3. Файл бази даних `library.txt` видаляється разом із робочою папкою додатка.
